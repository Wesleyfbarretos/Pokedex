## Getting Started

To run this project in the development mode is very simple.

### Installing

**Cloning the Repository**

```
$ git clone https://github.com/Wesleyfbarretos/portifolio-web.git

$ cd portifolio-web
```

**Installing dependencies**

```
$ yarn
```

_or_

```
$ npm install
```

### Running

```
$ yarn dev
```

_or_

```
$ npm run dev
```
